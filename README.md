## 📖 О проекте

Password Generation — это веб-приложение для создания криптографически стойких случайных паролей. 
Защищай свои аккаунты от взлома и утечек данных с удовольствием!

### 🌟 Особенности
- 🎨 **Магический дизайн** — лесная тематика с психоделическими грибами
- 🔐 **Безопасность** — генерация на стороне сервера, пароли не сохраняются
- 📱 **Адаптивность** — работает на телефоне, планшете и компьютере
- 🎯 **Гибкость** — настройка длины, типов символов, произносимые пароли
- 📋 **Удобство** — копирование в буфер, QR-код для переноса на телефон

---

## 🚀 Основной функционал

| Функция | Описание | Приоритет |
|---------|----------|-----------|
| 🔐 Генерация пароля | Создаёт случайный пароль заданной длины | Основной |
| 📏 Выбор длины | Настройка от 6 до 32 символов | Основной |
| 📋 Копирование | Мгновенное копирование в буфер обмена | Важный |
| 🎨 Смена фона | Атмосферные грибные обои (6 вариантов) | Дополнительный |
| 💬 Цитата о грибах | Вдохновляющая мудрость для души | Бонус |
| 🔤 Фильтры символов | Заглавные, строчные, цифры, спецсимволы | Основной |
| 🗣️ Произносимые пароли | Режим "слова" или "слоги" для лёгкого запоминания | Дополнительный |
| 📊 Индикатор надёжности | Визуальная оценка силы пароля | Важный |
| 📜 История | Последние 5 сгенерированных паролей (localStorage) | Дополнительный |

## 👤 Пользовательский сценарий
1.Пользователь открывает сайт
↓
→ Видит магический интерфейс
→ Читает цитату, замечает грибные обои
2.Настраивает длину пароля
↓
→ Двигает ползунок
→ Видит актуальное число
→ Понимает: "Могу сделать пароль нужной длины"
3.Нажимает "Создать пароль"
↓
→ Видит анимацию загрузки
→ Получает случайный пароль
→ Фон плавно меняется 🍄
4.Копирует пароль
↓
→ Нажимает "Копировать"
→ Видит подтверждение "✅"
→ Вставляет пароль где нужно
5.(Опционально) Генерирует ещё
↓
→ Если пароль не подошёл
→ Повторяет шаги 2-4
→ Наслаждается атмосферой ✨

## 🛠️ Технологический стек

### Backend
- **Python 3.12+**
- **Flask** — микрофреймворк
- **Werkzeug** (встроен в Flask) — WSGI сервер

### Frontend
- **HTML5** — семантическая разметка
- **CSS3** — анимации, градиенты, адаптивность
- **JavaScript (ES6+)** — Vanilla JS без фреймворков
- **Google Fonts** — Cinzel, Cormorant Garamond

### UI/UX & Ассеты
- Локальные изображения (`static/images/*.jpg`)
- CSS-переменные для лесной цветовой палитры
- Glassmorphism эффекты

### Инфраструктура & Инструменты
- **pip + venv** — управление зависимостями
- **JSON API** — обмен данными между клиентом и сервером
- **Browser Web API** — Clipboard API, localStorage
- **Docker** — контейнеризация
- **GitHub Actions** — CI/CD pipeline

## 📦 Установка и запуск

### 📋 Требования
- Python 3.10+ (проверка: `python3 --version`)
- Git (опционально)

### 🔧 Локальный запуск

#### Шаг 1: Клонирование (или скачивание)
```bash
git clone https://github.com/Moongrib/Password-Generation.git
cd Password-Generation

#### Шаг 2: Создание виртуального окружения
  # Linux/macOS
  python3 -m venv venv
  source venv/bin/activate

  # Windows
  python -m venv venv
  venv\Scripts\activate

#### Шаг 3: Установка зависимостей
  pip install flask

#### Шаг 4: Запуск сервера
  python app.py

  Ожидаемый вывод:
    * Serving Flask app 'app'
    * Running on http://127.0.0.1:5000

#### Шаг 5: Открыть в браузере
  Перейди по ссылке: http://127.0.0.1:5000


🐳 Запуск через Docker
Сборка образа
  docker build -t mushroom-pwd .
Запуск контейнера
  docker run -p 5000:5000 mushroom-pwd
Открой: http://localhost:5000
```
